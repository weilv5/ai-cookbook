## 📖 详细说明

### cloudflare-openai-gemini-proxy

**特点：**
- 基于 Cloudflare Workers，完全免费
- 同时支持 OpenAI 和 Google Gemini API
- 纯转发模式，API Key 由客户端携带，更安全
- 支持 CORS，完美适配 OpenClaw 模型配置
- 提供多种免费域名方案
